# 📚 Система управління бібліотекою

Консольна програма для автоматизації роботи бібліотеки. Лабораторна робота №2 з курсу "Git-технологія командної розробки проєктів".

## 📋 Опис проекту

Система дозволяє:
- Управляти книжковим фондом (додавання, видалення, пошук)
- Реєструвати та управляти читачами
- Видавати та приймати книги
- Відстежувати історію взяття/повернення книг

🚀 Встановлення та запуск
Вимоги
  .NET 6.0 SDK або вище
  Visual Studio 2026 (рекомендовано)
  Git

Кроки встановлення
1. Клонуйте репозиторій: git clone https://github.com/merialy/LibraryManagementSystem.git
2. Відкрийте Solution: LibraryManagementSystem.sln
3. Відновіть NuGet пакети: dotnet restore
4. Зберіть проект: dotnet build
5. Запустіть програму: cd Library, dotnet run

🧪 Тестування
Запуск всіх тестів: dotnet test
Запуск тестів з деталями: dotnet test --logger "console;verbosity=detailed"

Покриття тестами
- Book: 10 тестів, 100% покриття
- Reader: 8 тестів, 100% покриття
- LibrarySystem: 6 тестів, 100% покриття
- Загалом: 24+ тестів
